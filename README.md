## Credit Card Form

Set up using create-react-app

## Technology
* HTML
* JavaScript
* React
* CSS

### Setup Credit Card Form
* `git clone` the repo
* `cd` into credit_card_form
* `npm install` to add npm to your local file
* `npm start` to view credit_card_form on localhost:3000
